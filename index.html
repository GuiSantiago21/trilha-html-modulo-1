Como é uma estrutura básica de um HTML?

(Voltar)

Uma página HTML é formada obrigatoriamente por um conjunto de tags que definem a estrutura da página. A estrutura básica de um HTML é formado por:

DOCTYPE

O DOCTYPE não é uma tag HTML, mas uma instrução especial. Ela indica para o navegador qual versão do HTML deve ser utilizada para exibir a página.

HTML

Na estrutura do nosso documento, antes de começar a colocar o conteúdo, inserimos uma tag .

Head

A tag contém informações sobre o documento HTML que são de interesse somente do navegador e para outros serviços da web, e não para as pessoas que vão acessar nosso site.
São informações que não serão exibidas diretamente no navegador, também podemos considerar um local onde informamos os metadados sobre a página.

Body

A tag contém o corpo de um documento HTML, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Listas são muito úteis para ordenar itens que precisam ser listados como por exemplo uma receita de bolo ou um ranking de melhores filmes do ano

A diferença entre lista ordenada e lista não ordenada esta na forma como elas são apresentadas, uma lista organizada é apresentada com números ordenando os itens como 1,2,3,etc. Elas são feitas com a tag

    A lista não ordenada é mostrada com bullet points e pode ser utilizada para apresentação de itens de uma receita de bolo como falada acima, elas são feitas com a tag
